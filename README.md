# Smart Face Recognition Attendance System

Graduation Project

## Overview

A smart attendance management system that uses Face Recognition and QR Code technologies to automate attendance tracking in educational institutions.

## Technologies

### Mobile Application
- React Native

### Backend
- Laravel
- Laravel Sanctum
- REST API

### Database
- MySQL

### Face Recognition
- Face++ API

## Features

- User Authentication
- Face Registration
- Face Recognition Attendance
- QR Code Attendance
- Course Enrollment
- Attendance Sessions
- Attendance Reports
- Attendance Percentage Tracking
- Excel Report Export
